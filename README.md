## GithubAndroidClient 

GithubAndroidClient实现了的GitHub第三方Android客户端。 App的整体设计参考手机端Github H5效果。

目前主要实现功能：
1. 登录 
2. explore / showcases repos
3. profile / followers following
4. repo 
5. repo contents

这些数据是从GitHub提供的接口来的，地址是https://developer.github.com/v3/
